## Author: Bethany Feddes

## Requirements
JDK 8 or higher

##  Running the Program:
1. Open a terminal/command prompt.
2. Navigate to the directory containing Main.java on your machine.
3. To compile, enter: javac Main.java
4. To run the program, enter: java Main
5. Follow prompts.
